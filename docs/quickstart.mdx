## Prerequisites

- Node.js 18+
- MongoDB running locally or a remote connection string

## Installation

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/ContentsDigital-Lab/STD.Plus-ltd.git
    cd STD.Plus-ltd
    npm install
    ```
  </Step>
  <Step title="Configure environment">
    ```bash
    cp .env.example .env
    ```

    Edit `.env` with your values:

    | Variable | Default | Description |
    |---|---|---|
    | `NODE_ENV` | `development` | Environment mode |
    | `PORT` | `3000` | Server port |
    | `CORS_ORIGIN` | `*` | Allowed CORS origin |
    | `RATE_LIMIT_WINDOW_MS` | `900000` | Rate limit window (ms) |
    | `RATE_LIMIT_MAX` | `100` | Max requests per window |
    | `MONGODB_URI` | — | MongoDB connection string |
    | `JWT_SECRET` | — | Secret key for signing JWTs |
    | `JWT_EXPIRES_IN` | `1d` | JWT expiration time |
  </Step>
  <Step title="Seed the database">
    ```bash
    npm run seed
    ```

    This creates a default admin worker:
    - **Username:** `admin`
    - **Password:** `admin123`
    - **Role:** `admin`
  </Step>
  <Step title="Start the server">
    ```bash
    # Development (with auto-reload)
    npm run dev

    # Production
    npm start
    ```
  </Step>
</Steps>

## Scripts

```bash
# Reset the database (drops all collections)
npm run reset

# Seed the admin user
npm run seed

# Test RBAC on all endpoints
npm run test:rbac

# Test WebSocket events
npm run test:socket

# Test rate limiting
bash scripts/test-rate-limit.sh
```
